# Aplikasi Konverter Satuan (Latihan Android Studio)

Ini adalah aplikasi Android untuk mengonversi berbagai satuan pengukuran, termasuk luas, jarak, dan massa. Aplikasi ini menyediakan antarmuka yang sederhana dan ramah pengguna untuk melakukan konversi satuan dengan hasil yang detail.

## Fitur

- **Konversi Luas:** Mengonversi antara satuan luas yang berbeda
- **Konversi Jarak:** Konversi antara satuan jarak
- **Konversi Massa:** Konversi antara satuan massa
- **Konversi Suhu:** Konversi antara satuan suhu

## Teknologi yang Digunakan

- **Bahasa Pemrograman:** Java & Kotlin
- **Sistem Build:** Gradle
- **IDE:** Android Studio

## Cara Membangun

1. Kloning repositori ini.
2. Buka proyek di Android Studio.
3. Bangun dan jalankan aplikasi di emulator atau perangkat Android.
